Les textes

La taille

Il existe différentes façons de modifier la taille du texte en CSS, mais dans tous les cas, ce sera en utilisant la propriété font-size :

Taille absolue

p {
  font-size: 14px;
}

Dans ce cas-ci, la taille du texte a été modifiée en pixels, px. Il est possible d'utiliser des centimères (cm), ou des points (pt).

Appliquer une taille absolue n'est pas la meilleure des solutions parce qu'elle peut mal s'afficher selon la taille de l'écran.

Taille relative

p {
  font-size: 1.3em;
}

La taille relative est la plus adaptée pour les sites Web. Il existe une unité spécifique au CSS qui est le em.
1em équivaut à la taille normale de la police. 1.2em, par exemple, augmentera sa taille, tandis qu'à l'inverse, 0.8em la diminuera.
Notez bien qu'il faut utiliser un point et non une virgule pour séparer les décimales.

L'em n'est pas la seule unité qui existe pour définir une taille relative. Vous pouvez également utiliser le pourcentage ou les termes anglais :
xx-small, x-small, small, medium, large, x-large, xx-large, peu pratiques car il n'y a que 7 niveaux.

La police de caractère

Pour modifier la police de caractère d'un texte, il y a la propriété font-family :

p {
  font-family: police, "police secondaire", serif;
}

Lorsque l'on assigne une police de caractère à un texte, il ne faut pas oublier que si le visiteur ne possède pas la police en question sur sa machine, ça ne fonctionnera pas. A la place, c'est la police à défaut de son navigateur (différente d'un navigateur à l'autre et surtout, paramétrable !) qui s'affchera.

Il faut donc obligatoirement définir une des polices "standard", comme l'arial, le tahoma, georgia, verdana,...

Mais si je tiens vraiment à avoir une police de caractère originale sur mon site Web, comment faire ?
Bonne nouvelle, c'est à présent possible, mais il faut faire une petite manipulation supplémentaire : il va falloir charger la police de caractère en question.

@font-face {
  font-family: 'Mapolice';
  src: url('dossier/vers/mapolice.eot') format('eot');
}

/* maintenant qu'elle a été importée, je peux l'appliquer */
p {
  font-family: Mapolice, "Autre Police", serif;
}

Avec cette méthode il faudra évidemment que le fichier de la police se trouve dans le dossier renseigné. Et, pour ne pas nous faciliter la tâche, il faudra qu'elle y soit dans plusieurs formats et importées dans tous ces formats là également, pour un maximum de compatibilité entre les différents navigateurs.

Ca semble compliqué ? Par de soucis, il existe un site à garder dans vos favoris qui s'occupera de faire tout ça pour vous ! Il s'agit de Fontsquirrel.com

Pourquoi est-ce que plusieurs polices sont renseignées séparées par une virgule et qu'est-ce que c'est que ce serif ?

Il est possible, voire conseillé, de définir plusieurs police de caractères au cas où il y aurait un soucis avec la principale. Si Mapolice ne fonctionne pas, alors le navigateur appliquera Autre Police et, si ça ne fonctionne toujours pas, c'est n'importe quelle police serif qui sera utilisée.

Mais que signifie serif ?
serif défini toutes les polices avec empattements, telles que "Times New Roman", par exemple. Au contraire, sans-serif nous servira pour définir une police sans empattements, comme Verdana.

Italic, gras, souligné

Trois nouvelles propriétés qui sont :

font-style : normal, italic,... (pour l'italic)
font-weight : normal, bold (pour le gras)
text-decoration : underline, line-through, overline, blink, none (pour le souligné, barré, surligné, clignottant,...).

Alignement

Passons à la propriété text-align (left, right, center, justify).

p{
  text-align: right;
}

Attention, cette propriété ne fonctionne qu'avec les balises de type block.
Il n'est pas possible, par exemple de justifier un texte qui se trouve dans une balise de type inline comme <a>.
