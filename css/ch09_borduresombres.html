Bordures arrondies

Avant, dans le temps, à une époque très très lointaine, nous devions user de diverses ruses et subterfuges pour donner à nos paragraphes des coins arrondis.
Créer l'image dans photoshop, puis découper chaque coin séparément et s'arranger pour les placer juste comme il faut pour que ça passe inaperçu. Sachant qu'il ne "suffisait" pas non plus de créer un paragraphe avec plusieurs images de fond.

Aujourd'hui, il y a border-radius. Tout simplement.
Alors, comment fonctionne ce border-radius ?

p {
  border: 1px solid #000;
  border-radius: 20px;
}

Paragraphe avec une bordure d'1px à coins arrondis.

Afin de maximiser la compatibilité entre les différents navigateurs et leurs anciennes versions il est préférable d'ajouter les préfixes vendeurs :
-o- pour Opera
-moz- pour Gecko (Mozilla)
-webkit- pour Webkit (Chrome, Safari, Android...)
-ms- pour Microsoft (Internet Explorer)
-khtml- pour KHTML (Konqueror)

Dans le cas du border-radius, seuls les préfixes -moz- et -webkit- doivent être ajoutés.
N'hésitez pas à aller jeter un oeil sur caniuse.com si jamais vous n'êtes pas sûrs.

Concrètement :

p {
  border: 1px solid #000;
  border-radius: 20px;
  -moz-border-radius: 20px;
  -webkit-border-radius: 20px;
}

Vous pouvez ici aussi appliquer un border-radius différent à chaque coin (haut, droite, bas, gauche) :

p {
  border: 1px solid #000;
  border-radius: 20px 10px 15px 5px;
  -moz-border-radius: 20px 10px 15px 5px;
  -webkit-border-radius: 20px 10px 15px 5px;
}

Paragraphe avec une bordure d'1px à coins arrondis différents.

Et, encore plus loin, un arrondis en forme d'ellipse :

p {
  border: 1px solid #000;
  border-radius: 30px / 15px;
  -moz-border-radius: 30px / 15px;
  -webkit-border-radius: 30px / 15px;
}

Paragraphe avec une bordure d'1px à coins arrondis en forme d'ellipse.

Ombres

Ajouter une ombre à une "boîte", comme un paragraphe par exemple :

p {
  border: 1px solid #000;
  box-shadow: 4px 4px 3px #000;
  -webkit-box-shadow: 4px 4px 3px #000;
}

Paragraphe avec une ombre de 4px de décalage et de couleur noire.

Alors, que représentent ces différentes valeurs ?
1. Décalage horizontal
2. Décalage vertical
3. Adoucissement du dégradé (faible si inférieur au décalage, normal si égal, fort si supérieur)
4. La couleur

Pour ombrer les textes, la propriété est différente. Nous allons plutôt utiliser text-shadow, qui fonctionne avec les mêmes valeurs que box-shadow.

p {
  border: 1px solid #000;
  text-shadow: 2px 2px 2px #000;
}

Paragraphe avec un texte ombré.

Reconnue par tous les navigateurs sauf IE (seulement à partir de sa version 10), il n'est donc pas nécessaire d'ajouter un préfixe.
