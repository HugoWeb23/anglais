Les Liens

Un site Web n'est pas composé d'une seule page. Il en faut plusieures pour pouvoir prétendre avoir créé un site.
Comment relier ces pages les unes aux autres ? Comment faire pour créer un lien vers le site d'un ami ?

Le terme est lâché, il faut donc créer un lien. La balise pour les liens est, à la base, aussi simple que le <p> puisqu'il s'agit du <a>.
Je dis bien "à la base" parce que sans attributs, la balise <a> ne servira à rien.

Liens Externes

Voyons voir tout d'abord comment diriger les visiteurs vers un site autre que le nôtre. Cela s'appelera un lien externe.

<a href="http://www.technocite.be">Lien vers Technocité</a>

C'est donc l'attribut href qui contiendra comme valeur l'URL du site vers lequel diriger le visiteur. A noter qu'il faut l'adresse Internet complète en incluant le http://.
C'est ce que l'on appelera un lien absolu.

Liens Internes

Pour faire un lien interne, c'est à dire un lien qui nous emmènera de la page1 de notre site vers la page2, il faut de nouveau utiliser l'attribut href.

<a href="page2.html">Vers la page 2</a>

Ceci uniquement dans le cas où notre page2.html se trouve au même endroit, donc dans le même dossier, que la page1.html.
Ici ce n'est pas un lien absolu qui a été utilisé, mais plutôt un lien relatif. Bien qu'il soit possible d'écrire l'url en entier pour faire un lien interne, il est déconseillé de le faire.

Si votre page2.html se trouve dans un dossier différent de la page1.html, il va falloir adapter la valeur de l'attribut href.

Page2 située dans un dossier "enfant" de celui de la Page1

<a href="dossier/page2.html">Vers ma page 2</a>

Page2 située dans le dossier "parent" de celui de la Page1

<a href="../page2.html">Vers ma page 2</a>

Page2 située dans un dossier "parallèle" à celui de la Page1

<a href="../dossier/page2.html">Vers ma page 2</a>

Lien vers une ancre

Il peut arriver que le contenu d'une seule page soit tellement long qu'il faut beaucoup de temps pour remonter dans la page, ou pour arriver dans le bas.
Il est possible de créer un lien à l'intérieur d'une même page, toujours grâce à la balise <a>.

Pour cela, il va falloir préalablement donner un id à une balise qui servira de repère.

<h1 id="monancre">Titre de mon site<h1>

Plus loin dans la page, dans le bas par exemple, une fois que tout le texte a été lu, nous allons ajouter un lien qui va nous remonter vers notre repère :

<a href="#monancre">Haut de la page</a>

Si l'on désire faire un lien vers une ancre qui se trouve sur la page2.html, il faudra entrer le lien relatif vers la page en question suivi de l'ancre :

<a href="page2.html#monancre">Mon ancre sur la page 2</a>

Cas pratiques

Pour aller un peu plus loin, nous allons voir ce qu'il est également possible de faire avec la balise <a>.

Une infobulle

<a href="monlien.html" title="Infobulle">Mon lien</a>

La valeur de l'attribut title devrait contenir une description du lien. Le lien lui-même par exemple, ou un texte court définissant ce vers quoi mène le lien.
Ce sera très utile pour les synthétiseurs vocaux qu'utilisent les malvoyants et aveugles. Cela leur permettra de savoir vers où ils vont. De plus, les moteurs de recherche peuvent se baser sur ces valeurs pour référencer le site. Soyez donc précis et utilisez des mots clés adéquats.

D'accord, mais pourquoi parle-t-on d'une infobulle ?
Simplement parce que lors du passage du curseur de la souris sur le lien, une infobulle contenant la valeur de l'attribut title appraîtra.

Ouvrir un lien dans une nouvelle fenêtre / un nouvel onglet

<a href="monlien.html" target="_blank">Mon lien</a>

Nous ne pouvons pas forcer le lien à s'ouvrir dans une nouvelle fenêtre. Tout dépendra des paramètres de l'utilisateur. Dans Chrome, par exemple, target="_blank" ouvrira ma page par défaut dans un nouvel onglet.
