Mise en place du CSS

Ou écrire le CSS ?

Avant tout, un petit rappel de la différence entre le HTML et le CSS.

Comme dit précédemment, le HTML nous a servi à structurer notre page, à créer des titres, du contenu, à ajouter des images.
À présent, avec les CSS, nous allons pouvoir donner meilleure allure à tout ça en modifiant la couleur, en créant des blocs, des contours...

La première version des CSS (Cascading StyleSheet) est apparue en 1996. Mais alors, comment est-ce qu'ils faisaient pour changer la mise en forme de leur site à l'époque ?
Simplement, il existait des balises pour le faire. La balise <font> par exemple, qui maintenant n'existe plus, était utilisée avec différents attributs pour changer la taille, la couleur ou la police de caractère du texte.

Pourquoi est-ce que ça a été supprimé ? Parce que peu pratique.
Nous allons voir ça plus loin, avec les explications pour l'intégration des CSS.
Alors, où écrit-on ces fameux CSS ?

Dans un fichier .css à part

<!doctype html>
<html lang="fr">
  <head>
    <title>Titre de mon site</title>
    <link rel="stylesheet" href="css/monstyle.css" />
  </head>
  <body>
    <!-- contenu de mon site ici -->
  </body>
</html>

Donc, première solution, écrire le css dans un document séparé. Ici, appelé monstyle.css et placé dans un dossier css au sein de mon site.
Il nous suffit ensuite de le lier à notre page html, non pas avec la balise <a> vue précédemment, mais avec une autre balise spécifique à ajouter dans l'entête <head>.

Dans la zone de code ci-dessus, nous pouvons remarquer cette balise <link />.
Grâce à ce lien, notre document HTML sait qu'il doit intégrer les données affichées dans le document CSS qui se trouve dans le dossier css et qui se nomme monstyle.css.
Je peux lier de la sorte toutes les autres pages de mon site vers ce même fichier CSS.

Dans l'entête <head>

<!doctype html>
<html lang="fr">
  <head>
    <title>Titre de mon site</title>
    <style>
      p {
        color: #ff0;
      }
    </style>
  </head>
  <body>
    <!-- contenu de mon site ici -->
  </body>
</html>

Dans ce dernier exemple, une autre façon d'intégrer les CSS : ajouter une balise <style> dans l'entête <head> et y insérer les différentes règles CSS.
L'inconvénient de cette méthode, est que le style ne s'applique qu'à la page en cours. Il faut copier/coller toutes les règles dans toutes les autres pages de notre site Web.
Mais, c'est une solution qui peut être pratique dans le cas de la création d'une seule page.

Dans les balises grâce à l'attribut "style"

<p style="color:#ff0;" ><!-- texte ici --></p>

Cette dernière situation est ce qui représente le mieux ce à quoi ressemblait le code avant l'apparition des CSS. Pour chaque paragraphe, il fallait ajouter une balise, <font> par exemple, pour venir ajouter de la couleur à notre texte. Le gros problème avec cette méthode, c'est que non seulement il faut le mettre dans chaque page différente du site, mais en plus, si l'on désire mettre notre texte en bleu au lieu de rouge, il va falloir passer dans toutes les balises où une couleur a été appliquée et donc dans tous les paragraphes contenus dans chaque page de notre site Web.

Bien qu'elle soit la moins efficace dans le cas d'un site complexe, il ne faut pas oublier que cette méthode existe parce qu'elle sera très utile, voire l'unique solution, dans le cas de la création de mailings, à envoyer par courrier électronique.

Pour la suite du cours, il faudra donc retenir la première solution, celle qui consiste en la création d'un document .css séparé à lier à notre fichier HTML.

Sélectionner les balises

Passons dans notre document .css et voyons voir comment modifier la couleur de nos paragraphes par exemple :

p {
  color: #ff00ff;
}

Afin d'appliquer un style à nos paragraphes, donc à tout le texte qui est contenu dans les balises <p>, il suffit d'appeller notre balise en tapant tout simplement p.

Ensuite, c'est entre des accolades que nous allons ajouter des propriétés, comme la couleur (color), la taille du texte (font-size), suivie de la valeur que l'on souhaite lui donner.

Dans ce cas-ci, une couleur violette a été appliquée à nos paragraphes, grâce au code hexadécimal #ff00ff.

Le principe est donc simple :

Appliquer un style

balise 1 {
  propriété : valeur;
  propriété : valeur;
  propriété : valeur;
}

balise 2 {
  propriété : valeur;
}

Appliquer à plusieurs balises

h1, em {
  color : blue;
}

Tous les textes contenus entre les balises <h1></h1> et <em></em> seront de couleur bleue.

Insérer un commentaire dans un fichier CSS

/* Commentaire CSS */

Les class et id

Et si je ne veux pas que tous mes paragraphes soient identiques ?
Si je veux qu'un paragraphe au milieu de tout soit rouge et tous les autres bleus, comment faire pour isoler ce paragraphe là ?

Afin de cibler une balise en particulier, il est possible d'utiliser les attributs class et id.

Dans le document HTML :

<h1 id="montitre">Mon Titre</h1>
<p class="monparagraphe">Mon Texte</p>

Dans le fichier CSS :

#montitre {
  propriété : valeur;
}

.monparagraphe {
  propriété : valeur;
}

Mais dans quels cas utiliser class ou id ? Est-ce qu'ils ne font pas double emploi ?
Il faut savoir qu'un ID est unique, alors qu'une class peut être utilisée plusieurs fois dans la même page. Dans le cas d'une en-tête par exemple, vu qu'il n'y en aura qu'une seule, il serait préférable de lui attribuer un ID. Au contraire, pour les bloc de paragraphes qui se retrouveront un peu partout sur la page, il vaut mieux leur appliquer une classe.

Il existe deux balises dites universelles qui sont des balises "passe-partout". Elles ont été créées sans propriétés par défaut pour que l'on puisse les utiliser dans des situations précises en les modelant à notre convenance.

Ces balises sont :

<div></div> : balise de type block, voisine des balises <p> et <h1>.
<span></span> : balise de type inline, comme les balises <strong> ou <em>.

Pour mieux comprendre à quoi elles servent, voyons un exemple :

<p>Bonjour et bienvenue sur mon site Web !</p>

J'aimerais mettre en couleur rouge le mot bienvenue, comment faire ?

<p>Bonjour et <span class="rouge">bienvenue</span> sur mon site Web !</p>

J'ai sélectionné le mot bienvenue, mais pour le moment, aucun changement n'est visible sur mon site. Pour cela, il va falloir passer dans le fichier CSS et ajouter une propriété de couleur ayant pour valeur rouge à notre classe "rouge".

.rouge {
  color: #ff0000;
}

A présent, nous pouvons constater que le mot a bien été isolé du reste et qu'il s'affiche en rouge.

Bonjour et bienvenue sur mon site Web !

Sélecteurs avancés

Voici quelques autres sélecteurs et d'autres façon de sélectionner les balises ou éléments qui se trouvent dans le document HTML :

* { } /* Sélectionne toutes les balises */
p em { } /* Sélectionne les balises <em> qui se trouvent dans les balises <p> */
p + a { } /* Sélectionne la première balise <a> qui suit une balises <p> */
a[title] { } /* Sélectionne les <a> qui ont un attribut title */
a[title="valeur"] { } /* Sélectionne les <a> dont la valeur de title est valeur */
a[title*="mot"] { } /* Sélectionne les <a> dont la valeur de title contient mot */
