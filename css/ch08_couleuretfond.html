Couleurs et fonds

Couleur du texte

Nous avons déjà fait connaissance avec la propriété color qui sert à changer la couleur du texte.
Il existe plusieurs façon de modifier cette couleur : grâce à son nom, grâce à un code hexadécimal, ou enfin, avec le code RGB.

h4 {
color: red; /* mon titre sera rouge */
}

p {
color: #00ff00; /* mon paragraphe sera vert */
}

a {
color: rgb(0,0,255); /* mon lien sera bleu */
}

Couleur de fond

La propriété à utiliser pour modifier la couleur de fond est background-color. Elle fonctionne de la même façon que color, il suffit de lui ajouter comme valeur le nom d'une couleur, le code hexadécimal ou la valeur RGB.

body {
  background-color: #ffff00; /* La couleur de fond sera jaune */
}

Losque vous appliquez un style au body, il s'appliquera à toute la page. C'est le fond de votre page HTML qui sera jaune dans ce cas-ci.

Cliquez ici pour voir l'exemple sur une page entière.

Lorsque l'on défini une valeur pour une propriété dans body, elle sera appliquée pour tout le document. Par exemple si, dans mon document CSS, je dis que body doit contenir du texte rouge, toutes les balises contenues dans <body> hériterons du texte rouge sauf si elles ont un style qui est déjà défini.
C'est la déclaration la plus précise qui "gagne".

Images de fond

La couleur de fond, c'est bien, mais les images, c'est mieux ?
Pas de problème, il est possible grâce à la propriété background-image de mettre une image de fond à un site Web.

A vrai dire, il est même possible d'ajouter une image de fond à un paragraphe aussi, voire à un lien. Comme pour la couleur de fond.

p {
  background-image : url('../images/pattern.gif');
}

Mon texte dans un paragraphe qui a un background-image

A partir de cette image de fond, plusieurs paramètres peuvent être modifiés.

background-attachment:
fixed indique que le fond doit rester fixe et ne dois donc pas défiler avec le texte lorsque l'on descend dans la page.
scroll, au contraire, le fait défiler.

background-repeat :
no-repeat, repeat-x, repeat-y, repeat, précisent si le fond doit être répété verticalement, horizontalement, entièrement ou pas du tout.

background-position : horizontal vertical
"horizontal" sur la ligne précédent doit être remplacé par l'une des valeurs suivantes : left, right, center.
"vertical" doit quant à lui être remplacé par : top, bottom.
Ou tout simplement, l'un et l'autre, peuvent contenir des valeurs en px ou en %.

Toutes ces infos peuvent être mise sur une même ligne !

p {
  background-color: #dd9;
  background-image: url('../images/pattern.gif');
  background-repeat: no-repeat;
  background-position: right 20px;
}

Peut s'écrire :

p {
  background : #dd9 url('../images/pattern.gif') no-repeat right 20px;
}

Voyons ça de plus près :
Nous avons donc un paragraphe, auquel nous avons appliqué une image de fond qui ne se répète pas, qui doit scroller avec le texte, et qui est positionnée à droite et à 20px (à partir du haut !). Nous lui avons aussi mis une couleur de fond qui s'affiche là où il n'y a pas d'image.

Mon texte dans un paragraphe qui a un background défini.

Mais ce n'est pas terminé. En effet, depuis CSS3, nous avons la possibilité d'afficher plusieurs fonds. Il suffit de les séparer à l'aide d'une virgule, et ce sera le fond déclaré en premier qui se trouvera le plus audessus.

En prenant une image représentant un paysage et une autre un soleil, voilà ce que l'on peut obtenir :


background : url('../images/soleil.png') no-repeat 95% 10%, url('../images/ciel.jpg') scroll;

Un joli soleil sur un ciel bleu.

Transparence

Il est dorénavant possible d'ajouter de la transparence à des paragraphes, textes, ou autres éléments HTML en modifiant son opacité.
Deux propriétés sont disponibles pour ça :

opacity: 1=opaque, 0=transparent, 0.6=transparent à 60%,... etc.
background-color: rgba(0,0,0,0.5). Le 0.5 correspond à la valeur de l'opacité.

La différence entre les deux ?
Si vous utilisez opacity sur un élément de page, tout son contenu sera transparent.
Si au contraire, vous voulez simplement lui donner un fond transparent, alors là il faudra utiliser le rgba.

Etant donné que cette notion n'est compatible avec IE que depuis sa version 9, il est préférable d'ajouter un ligne avec le rgb classique.
