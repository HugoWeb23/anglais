Comportement dynamique

Au survol

Un petit chapitre cette fois-ci pour passer en revenue quelques comportements dits "dynamiques" que l'on peut créer grâce au CSS.

Vous vous êtes peut-être demandé, lorsque l'on a parlé des liens, comment faire pour que la couleur du lien change.
Nous allons pour cela utiliser ce que l'on appelle des "pseudo-classes".

Pour le survol, on utilise la pseudo-classe :hover.

a {
  color: #0f0;
}

a:hover {
  color: #f00;
}

Il faut d'abord définir la valeur par défaut du a et ajouter une autre règle pour le :hover.

Cette pseudo-classe fonctionne aussi sur des paragraphes, des titres,...

Au clic

Pour le clic, on utilise la pseudo-classe :active.

a {
  color: #0f0;
}

a:active {
  color: #ff0;
}

Cette pseudo-classe est utilisée principalement pour les liens. L'effet est très rapide puisqu'il s'applique au moment même du clic.

À la sélection

Pour la sélection, on utilise la pseudo-classe :focus.

a {
  color: #0f0;
}

a:focus {
  color: #0ff;
}

Cette pseudo-classe sera surtout utilisée pour les champs de formulaire. Elle permettra de mettre en surbrillance le champs actuellement actif.

Lien visité

Pour signifier qu'un lien a été visité, on utilise la pseudo-classe :visited. Par défaut, les liens visités sont affichés en mauve.

a {
  color: #0f0;
}

a:visited {
  color: #0ff;
}

Encore une fois cette pseudo-classe sera surtout utilisée pour les liens.
